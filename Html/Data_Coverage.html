def perc_global(data, how='area'):
    
    if how is 'area':
        element = 'Area harvested'
    
    elif how is 'production':
        element = 'Production'
    
    tmp = fao.copy()
    tmp1 = tmp[(tmp['Element'] == element)]
    tmp1 = tmp1.sort_values(['Country', 'Item Code', 'Year'])

    multi_index = pd.MultiIndex.from_product([tmp1['Country'].unique(),
                                              tmp1['Item Code'].unique(),
                                              tmp1['Year'].unique()],
                                             names=['Country', 'Item Code', 'Year'])

    tmp1 = tmp1.set_index(['Country', 'Item Code', 'Year']).reindex(multi_index).reset_index()
    tmp1 = tmp1.set_index(['Country', 'Item Code', 'Year'])
    tmp1 = tmp1.interpolate(method='linear',
                              axis=0,
                              limit_direction='both')
    tmp1 = tmp1.reset_index()
    tmp1 = tmp1[(tmp1['Year'] == float(avg_year))]
    tmp1['overlap1'] = np.in1d(tmp1['Country'], df.NAME_0.unique())
    tmp1['overlap2'] = np.in1d(tmp1['Item Code'], df.Item_Code.unique())

    data_in = tmp1.query("overlap1 == True & overlap2 == True")['Value'].sum()
    data_out = tmp1['Value'].sum()
    perc_rep = round(100 * (data_in / data_out), 2)
    
    return perc_rep

Data


General

  • Our dataset caputres the amount crop production by farms size
  • 55 countries are represented at either the national or subnational level.
  • In total, there are 2800 national or subnational units.
  • This captures 13.8% of global harvest area and 18.52% of global crop production
  • There are 126 crops which we were able to match with the FAO's database to calculate the amount of crops produced by farm size class for food, feed, waste, seed, proccessing, and other in terms of kcal.
  • We used 36 tabulated datasets, and 19 microdatasets (i.e., data at the household record level)
  • 40 agricultural censuses were used. Where census data was not used, nationally or subnationally representative household surveys were used (15 in total).
  • On average the data was from 2011, with the oldest datasets from 2001 and the newest from 2013
  • We used the World Census of Agriculture's (WCA) farm size categories to be consistent with other studies.
